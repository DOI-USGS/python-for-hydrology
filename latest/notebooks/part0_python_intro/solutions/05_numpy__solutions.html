TEST YOUR SKILLS #1¶

Let’s calculate the amount of material lost from Mt. St. Helens in the 1980 eruption. We have the before and after elevation arrays in the data folder noted in the next cell. We will need to load them up. - The before and after files are the same dimensions, but they have some zeros around the edges indicating “nodata” values. - Check to see - are the zeros in the same place in both arrays? (np.where and friends can help with that). - If there are no-data zeros in one array at a different location than the other array, what happens if you look at the difference? - assume each pixel is 25 x 25 feet and the elevation arrays are provided in meters to calculate a volumetric difference between before and after arrays.

[10]:

before_file = datapath / 'mt_st_helens_before.dat'
after_file = datapath / 'mt_st_helens_after.dat'

[11]:

before = np.loadtxt(before_file)
after = np.loadtxt(after_file)
# convert from meters to feet (!)
before *= 3.28084
after *= 3.28084

[12]:

np.sum((before-after) * 25 * 25)

# we need to find the values where EITHER after is 0 or before is 0 and deal with them somehow
